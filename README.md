# RedPixelDream

**RedPixelDream** is a minimalist and functional screensaver (DreamService) for Android, designed specifically for nighttime use. The application focuses on readability, low eye strain, and protecting OLED/AMOLED displays from burn-in.

---
### 🤖 About the project
 
*This project it was built in **Android Studio**  human developer and the **Gemini** AI model.*
---

## 🌟 Features (V1.0 Stable)

- **Design**: 
  -.
  - **EN:** Deep red clock with a premium outline effect for minimal screen glow.
  - **EN:** Automatically wakes up when proximity is detected (hand wave).
  - **EN:** Event list (today/tomorrow) + graphical monthly grid.
  - **Battery Protection**: 
  - **EN:** Visual warning (80% LIMIT) to protect battery health.
  - **Burn-in Protection**: 
  
  - **EN:** "Pixel Shift" system to prevent image retention on AMOLED screens.
  - **Ultra-low Brightness**: 
  - **EN:** Automatic 0.03f brightness for perfect night comfort.

## 🚀 How to use?

1. **Installation**: Compile in Android Studio & install on your device.
2. **Permissions**: 
   - Grant **Calendar** access.
   - Enable **"Display over other apps"** in Android Settings.
3. **Activation**: 
   - Settings -> Display -> Screen Saver -> Select **Czerwony Pixel**.
